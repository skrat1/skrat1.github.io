Microsoft Windows

Pojem "Windows" je v informatice označení pro několik různých operačních systémů vytvořených firmou Microsoft. Jednotlivé operační systémy se od sebe liší jádrem systému, podporou multitaskingu, účelem použití a užívanými knihovnami.

Název "Windows" pochází z jeho schopnosti otevírat a používat více apliikací najednou, tak aby každá běžela ve svém vlastním okně.

První verze Windowsu byla na trh uvedena v roce 1985 pro systém MS-DOS s název , ale hromadného rozšíření na trhu se dočkala až verze Windows 95.

V dnešní době se nejvíce použíbá verze Windows 8.1 a Windows 10, nebo pro serverové verze to jsou Windows Server 2008, 2012, 2016 a 2019.

Za posledních 35 let vydal Microsoft několik různých druhů operačního ystému Windows.

Windows pro DOS

První na trh,jak už jsme zmínili, se dostal Windows 1.0, který byl uveden 20.listopadu 1985. Dokázal spuštět většinu jednoduchých programů, které byli původně určené pro systé DOS.

Jako další verze se na trh dostala veze Windows 2.0. Vyšla v listopadu 1987 a od se lišila několika vylepšeními správy paměti a uživatelského rozhraní.

Prvními systémi, co dosáhli nějakého komerčního úspěchu bylu Windows 3.0,který vyšel na trh 22.května 1990, a Windows 3.1, který 18. března 1992. Jejich systémové jádro obsahovalo částečnou podporu 32 bitových ovladačů, přistupujících k hardwaru přímo a to bez použití Dosu a BIOSu, jak tomu bylo dříve.

Vzhledovou změnu v grafickém rozhraní obsahoval Windows 95, který se dostal na trh 24.srpma 1995. Tato změna přinesla lepší ovládání a větší zájem ze strany uživatelů. Jen za první den se prodalo 300 000 kopií.

Jako další verze se dostala na trh 25.června 1998 verze Windows 98 s přidanou podporou USB, Internet Explorerem 4.01 a dalšími vylepšeními. V květnu 1999 vyšla druhá verze, Windows 98 Second Edition, opravující menší chyby, navíc obsahovala i Internet Explorer 5.0 a Windows Media Player 6.2.

14.září 2000 byla na trh uvedena verze Windows Millennium Edition, poslední z řady systémů pro DOS.

Windows New Technology

V řadě systémů Windows NT byla jako první verze vydaná v roce 1993 s názvem Windows NT 3.1. Následně v roce 1994 vyšla verze Windows NT 3.5 a rok na to verze Windows NT 3.51.

Mezi další populární verze například patří Windows 4.0 z roku 1996, Windows 2000 z roku 2000, Windows XP z roku 2001, Windows XP x64 z roku 2005, Windows Vista z roku 2007, Windows 7 z roku 2009 a nebo Windows 10 z roku 2015.
