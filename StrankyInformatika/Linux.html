Linux

Linux je v soucasnosti nejrozšířenější Open Source operační systém. Jádro tohoto operačního systému - Linux kernel - začal psát v roce 1991 finský student Linus Torvalds, ale v současné době se na jeho vývoji podílí tisíce lidí z celého světa.

Jádro používá licenci GPL2, která vyžaduje, aby ten, kdo v něm něco změní a změněné jádro předává zdarma či za peníze ostatním, poskytl svým uživatelům i zdrojové kódy. Jádro podporuje desítky různých architektur procesorů.

Toto jádro je základem jednak stovek Linuxových distribucí a jednak operačního systému Android. Z komerčních distribucí jsou nejčastěji používané Red Hat Enterprise Linux, SUSE Linux Enterprise Server či Ubuntu LTS. Z nekomerčních distribucí je nejčastěji používano Ubuntu, dále Debian, Fedora, OpenSUSE, Mint, Arch a mnoho dalších. Linux je používán na naprosté většině světových superpočítačů. Mnoho základních programů v distribucích Linuxu pochází z projektu GNU. Jako grafické rozhraní je používán buď X server nebo Wayland a s ním jako nejčastější sada knihoven GNOME nebo KDE. Jako systémový překladač pro C a C++ je v Linuxu používáno GCC nebo v některých distribucích LLVM a Clang, mnoho programů je ale interpretovaných, napsaných třeba v jazycích Python a Perl.
