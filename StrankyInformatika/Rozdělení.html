Rozdělení a Terminologie

Software a data

V počítači vše co není hardware je software. Označením software však většinou zahrnuje pouze programy v počítači, ale tento pojem se může vztahovat i na data, která popisují například obrázek, textový dokument a další dokumenty

Malware, neboli škodlivý software

Malware je program určený k proniknutí do počítačového systému či k poškození. Jsou to virové programy.

Ransomware, neboli vyděračský software

Program požadující po oběti výkupné za obnovení přístupu, který program oběti odebral blokováním počítačových systémů, či zašifrováním dat.

Spyware, neboli špehovací software

Program využívající internetových stráne k odeslání dat z uživatelova počítače bez jeho vědomí.

Adware, neboli reklamní software

Program znepříjemňující práci reklamní aplikací.

Crapware

Crapware je nechtěný, ale předem nainstalovaný software v počítači.
